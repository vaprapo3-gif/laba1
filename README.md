# Лабораторна робота №1
## Тема: Проектування програм обробки одновимірних масивів

### Варіант 8
Підрахувати середнє арифметичне всіх елементів у даному одновимірному масиві.

## Короткий опис задачі
Програма вводить одновимірний масив цілих чисел, обчислює середнє арифметичне всіх його елементів, а також виконує сортування елементів масиву:
- елементи з парними індексами сортуються за зростанням;
- елементи з непарними індексами сортуються за спаданням.

## Опис алгоритму розв’язання
1. Ввести кількість елементів масиву.
2. Ввести елементи масиву.
3. Обчислити суму всіх елементів масиву.
4. Знайти середнє арифметичне за формулою:
   average = sum / n.
5. Виконати сортування:
   - для парних індексів — за зростанням;
   - для непарних індексів — за спаданням.
6. Вивести середнє арифметичне.
7. Вивести масив після сортування.

## Пояснення основних етапів роботи програми
Спочатку користувач вводить кількість елементів масиву та самі елементи. У процесі введення одночасно обчислюється сума елементів. Після цього знаходиться середнє арифметичне всіх елементів масиву.

Далі виконується сортування масиву. Елементи, що стоять на парних індексах, впорядковуються за зростанням, а елементи, що стоять на непарних індексах, — за спаданням.

Після завершення роботи програма виводить середнє арифметичне та відсортований масив.

## Висновок
У ході виконання лабораторної роботи було створено програму для обробки одновимірного масиву. Програма обчислює середнє арифметичне всіх елементів масиву та виконує сортування відповідно до умови завдання. Отримані результати відповідають поставленій задачі.
